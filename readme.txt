
CMS сделанная с использованием php, mysql, javascript, css, html, jQuery, Ajax
версия: 3.0
краткое опиание структуры каталогов и некоторых файлов:
	"ajax/" - каталог содержащий файлы обработчики для Ajax вызовов
	"app/" - основной каталог, содержащий ядро, настройки, шаблоны и модули
		"classes/" - каталог общих классов
			"classes/Main_classes.php" - файл класса, содержащий основные методы для работы с mysql запросами, транслитерации
			"classes/User.php" - файл класса, содержащий основные методы для работы с пользователями
		"core/" - каталог ядра
			"core/controller.php" - главный Контролер 
			"core/dbo.php" - файл класса для работы с БД
			"core/model.php" - главная Модель
			"core/module_router.php" - файл класса minc (роутер для модулей)
			"core/router.php" - основной роутер
			"core/sessions.php" - файл для работы с сессиями пользователей
			"core/view.php" - главный Вид (файл класса, отвечающий за выбор шаблонов, а также создание новых "дефальтных", если запрашиваемые файлы шаблонов не находит система)
		"includes/" - каталог дополнительных файлов, в данный момент не используется
		"modules/" - каталог модулей
			"modules/breadcrumbs/" - модуль "хлебных крошек"
			"modules/elem/" - модуль по выводу отдельных елементов (заморожен из за ненадобности)
			"modules/form/" - модуль форм, их построению и обработке
			"modules/html/" - модуль по выводу html блоков (заморожен из за ненадобности)
			"modules/login/" - модуль авторизации
			"modules/meditor/" - модуль галереи фотографий (заморожен)
			"modules/menu/" - модуль построения меню
			"modules/pagination/" - модуль пагинации
			"modules/register/" - модуль регистрации
			"modules/shortlink" - модуль коротких ссылок
		"templates/" - каталог шаблонов
			"templates/administrator/" - каталог стандартного шаблона административной панели сайта
			"templates/default/" -  каталог стандартного шаблона сайта
			"templates/html/" - каталог "кастомных" шаблонов для отдельных элементов сайта
		"bootstrap.php" - файл, отвечающий за общие настройки сайта, а также связывающий систему воедино
	"css/" - каталог CSS файлов
	"fonts/" - каталог шрифтов, используемых на сайте
	"images/" - каталог изображенний, используемых при построении страниц
	"js/" - каталог javascript файлов и jquery плагинов и библиотек
	"uploads/" - каталог медиа файлов пользователей
	".htaccess" - локальный конфигурационный файл вебсервера
	"404.php" - страница 404 ошибки
	"index.php" - входной файл сайта

Установка:
	1) закачиваем файлы на сервер
	2) открываем файл "core/dbo.php" и прописываем настройки для подключения к БД	
		private $_connect = array(
			'host' => 'localhost',
			'port' => '3306',
			'user' => '**********', //имя пользователя базы данных
			'pass' => '**********', //пароль пользователя базы данных
			'name' => '**********', //имя базы данных
			'charset' => 'utf8',
			'debug' => true
		);
	3) импортируем файл дамп формата .sql (лежит в корневом каталоге, содержит тестовую информацию)
	4) смотрим рабочий сайт
